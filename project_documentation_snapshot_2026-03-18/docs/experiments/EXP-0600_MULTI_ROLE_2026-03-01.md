# EXP-0600: Multi-Role Evolution (Phase 4) — 2026-03-01

## Background

Phase 4 tests collective fitness and specialization. Organisms with different strategy profiles cooperate on multiple wiki-derived micro-tasks. Phase 3 energy model was closed as FAIL (ADR-0009); Phase 4 uses Phase 2 penalty regime (Regime B, λ=0.01).

## Hypotheses & Relation to CANON/ADR

- **H6 (implicit):** Collective performance can exceed best single-role individual when roles specialize.
- **ADR-0009:** Phase 4 proceeds without energy model; penalty regime from ADR-0005.
- **ADR-0007:** Formalized canonical collective-advantage gate metrics (A1/A2/A3) and retired the biased old comparator.

## Protocol

- **Micro-tasks:** Cloze (existing) + Entity detection (binary, Title Case heuristic).
- **Collective fitness:** `weighted_mean(best_cloze, best_entity)`.
- **Individual fitness:** `max(cloze_acc, entity_acc)` (Scheme S1, specialist-friendly).
- **N=30, G=100, P=200**, subset_size=100, simplified_wiki_v0.
- **Genome:** ClozeGenome v2, Regime B, speciation active.
- **Energy model:** Off.

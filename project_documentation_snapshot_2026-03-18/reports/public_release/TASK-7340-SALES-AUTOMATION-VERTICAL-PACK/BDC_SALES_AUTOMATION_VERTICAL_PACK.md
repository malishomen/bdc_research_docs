# BDC Sales Automation Vertical Pack

## Vertical Position
BDC Designer helps sales teams select and refine cooperative AI workflow architectures for lead triage, qualification, proposal generation, and review gates.

## Target Use Cases
- Lead triage -> qualification -> proposal -> risk review
- CRM response -> offer tailoring -> approval gate
- Outbound personalization -> verifier check -> final send gate

## Typical Pilot Structure
1. Baseline freeze for one workflow.
2. BDC CLI recommendation and caution review.
3. Hybrid refinement under matched budget.
4. Baseline vs BDC vs hybrid comparison.
5. Conversion and delivery proof package.

## Scope Safety
Restricted BDC scope applies; no universal optimization claims.

# BDC Commercial Pilot Kit

## Offer Position
Restricted BDC pilot engagements focus on measurable architecture-design improvements with explicit scope boundaries.

## Pilot Scope
- One selected workflow per pilot cycle
- Baseline freeze + descriptor capture
- CLI recommendation + hybrid refinement
- Baseline vs BDC vs hybrid outcome package

## Deliverables
- Pilot kickoff plan
- Baseline freeze artifact set
- Recommendation trace
- Refinement trace
- Final outcome report and next-step decision

## Pricing Bands
- Starter Pilot: fixed scope, 2-3 weeks
- Standard Pilot: broader instrumentation, 4 weeks
- Advanced Pilot: multi-stakeholder and governance integration, 6 weeks

## Success Conditions
- Time-to-target improvement
- Quality improvement
- Search-cost and/or iteration reduction
- Complete reproducible artifact trail

## Scope Safety
No universal optimization claims. Outcomes are evaluated within Restricted BDC v1 boundaries.

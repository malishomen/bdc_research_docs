## Часть I. Ключевые факты из исследования

### 1.1 Фундаментальные константы

| Константа | Значение | Определение |
|---|---|---|
| φ (золотое сечение) | 1.6180339887... | Положительный корень x² - x - 1 = 0 |
| φ⁻¹ | 0.6180339887... | φ - 1 = 1/φ |
| φ⁻² | 0.3819660113... | 1 - φ⁻¹ |
| Φ (Knuth 32-bit) | 2654435761 | ⌊2³² / φ⌋ + 1 |

Свойства:
- φ = 1 + 1/φ (единственное такое положительное число)
- lim(F(n+1)/F(n)) = φ при n → ∞
- φ — «самое иррациональное» число: наихудшая аппроксимация рациональными дробями
- Связь с пентагональной/декагональной симметрией

### 1.2 B-ДНК: числовые пропорции

| Параметр | Значение | Фибоначчи / φ соответствие |
|---|---|---|
| Шаг спирали | 34 Å | F(9) = 34 |
| Диаметр спирали | 21 Å | F(8) = 21 |
| 34 / 21 | 1.619047... | ≈ φ, отклонение < 0.1% |
| Большая бороздка | 21 Å | F(8) = 21 |
| Малая бороздка | 13 Å | F(7) = 13 |
| 21 / 13 | 1.615384... | ≈ φ, отклонение < 0.2% |
| Пар оснований на виток | 10 | Декагональная симметрия C₁₀ |
| Угол поворота | 36° | 360° / 10; связан с пентагоном |
| Подъём на пару | 3.4 Å | 34 / 10 |

### 1.3 Нуклеосомная упаковка

| Параметр | Значение | Фибоначчи-связь |
|---|---|---|
| Длина ДНК на нуклеосому | 147 bp | ≈ F(12) = 144, отклонение 2% |
| Число витков вокруг гистона | 1.65–1.7 | ≈ φ |
| Паттерн 10n | Компактный гетерохроматин | Неактивные гены |
| Паттерн 10n+5 | Гибкий эухроматин | Активные гены |

### 1.4 Геномные «супракоды» (Перес)

- 64 кодона распределены вокруг φ-аттракторов
- Резонансы: сегменты длиной F(n) содержат F(n-1) нуклеотидов одного типа
- Таблица генетического кода = матрица контрольных сумм
- При отклонении пропорций > ~0.1% — возможен триггер апоптоза

### 1.5 Критический анализ (из исследования)

- Параметры B-ДНК варьируются от условий среды (GC-состав, гидратация)
- φ в дискретной системе атомов — всегда рациональное приближение
- Статистический шум: при миллиардах bp вероятность «случайных» φ-совпадений ненулевая
- «Фрактальная антенна» / биофотоны — высокоспекулятивно, не инженерная гипотеза

**Вывод исследования:** φ — вероятный «путь наименьшего сопротивления» для организации материи, не «цель» эволюции.

## Часть II. Текущие применения φ / Fibonacci в BDC

### 2.1 Константы (`evolution/cloze_symbolic/fibonacci.py`)

```python
PHI_INV = (5.0**0.5 - 1.0) / 2.0          # = 0.6180339887...
GOLDEN_RATIO_32 = 2654435761               # Knuth multiplicative hash constant
```

**Математика:**

\[
\varphi^{-1} = \frac{\sqrt{5} - 1}{2} \approx 0.6180339887
\]

\[
\Phi_{32} = \lfloor 2^{32} / \varphi \rfloor + 1 = 2654435761
\]

### 2.2 Fibonacci-хеширование (`fibonacci_hash_pair`)

**Файл:** `evolution/cloze_symbolic/fibonacci.py`

```python
def fibonacci_hash_pair(a: int, b: int, table_size: int) -> int:
    key = (int(a) ^ int(b)) & 0xFFFFFFFF
    mixed = (key * GOLDEN_RATIO_32) & 0xFFFFFFFF
    frac = ((mixed * PHI_INV) % 1.0)
    return int(frac * table_size)
```

**Математическая формула:**

\[
h(a, b) = \lfloor \text{table\_size} \cdot \text{frac}\left( \frac{((a \oplus b) \cdot \Phi_{32}) \mod 2^{32}}{2^{32}} \cdot \varphi^{-1} \right) \rfloor
\]

Где:
- \( \oplus \) — побитовое XOR
- \( \Phi_{32} = 2654435761 \) — константа Кнута
- \( \text{frac}(x) = x - \lfloor x \rfloor \) — дробная часть
- φ⁻¹ обеспечивает равномерное распределение (свойство «самого иррационального» числа)

**Назначение в BDC:** Равномерное распределение пар (prev_token, candidate) по бакетам контекстной таблицы в ClozeGenomeV1. Используется для хеширования в таблицы размером 8 и 5 (числа Фибоначчи).

**Связь с исследованием:** Свойство φ как «самого иррационального» числа гарантирует минимальную кластеризацию при малых таблицах. Это прямое инженерное использование математического свойства, описанного в Части I.1.

### 2.3 Fibonacci-размеры таблиц (`genome.py`, v1)

**Файл:** `evolution/cloze_symbolic/genome.py`, `ClozeGenomeV1`

| Таблица | Размер | Число Фибоначчи |
|---|---|---|
| `prev_context_table` | 8 | F(6) = 8 |
| `next_context_table` | 5 | F(5) = 5 |

**Обоснование:** Размеры Фибоначчи для хеш-таблиц минимизируют коллизии при использовании Fibonacci-хеша, так как φ⁻¹ × F(n) имеет наилучшую равномерность по F(n) бакетам.

**Параметрический бюджет:**
- `top_k=5`: 1 + 5 + 8 + 5 = **19 параметров**
- `top_k=8`: 1 + 8 + 8 + 5 = **22 параметра**

### 2.4 φ-масштабированные мутации (`mutate.py`)

**Файл:** `evolution/cloze_symbolic/mutate.py`

```python
sigma_a = sigma_base * (PHI_INV ** 2)  # ≈ 0.382 * sigma_base
sigma_t = sigma_base * PHI_INV         # ≈ 0.618 * sigma_base
sigma_g = sigma_base * 1.0             # = 1.000 * sigma_base
```

**Математическая иерархия:**

\[
\sigma_A = \sigma_0 \cdot \varphi^{-2} \approx 0.382 \cdot \sigma_0
\]
\[
\sigma_T = \sigma_0 \cdot \varphi^{-1} \approx 0.618 \cdot \sigma_0
\]
\[
\sigma_G = \sigma_0 \cdot \varphi^{0} = 1.0 \cdot \sigma_0
\]

Каждый acid получает мутацию `N(0, σ)`:

\[
w' = \text{clip}(w + \mathcal{N}(0, \sigma_{\text{acid}}),\ -5,\ 5)
\]

**Биологическая аналогия из исследования:**
- `Acid_A` (частота) — самый консервативный канал, аналог стабильного backbone
- `Acid_T` (прямой контекст) — промежуточная изменчивость
- `Acid_G` (обратный контекст) — самый пластичный канал

Это напрямую отражает иерархию стабильности ДНК: водородные связи (консервативные) vs. стекинг-взаимодействия (пластичные).

**Отношения между уровнями:**

\[
\frac{\sigma_T}{\sigma_A} = \frac{\varphi^{-1}}{\varphi^{-2}} = \varphi \approx 1.618
\]
\[
\frac{\sigma_G}{\sigma_T} = \frac{1}{\varphi^{-1}} = \varphi \approx 1.618
\]

Каждый следующий acid ровно в φ раз более мутабелен — точная φ-лестница.

### 2.5 Golden Section Selection (`select.py`)

**Файл:** `evolution/cloze_symbolic/select.py`

```python
GOLDEN_ELITE_FRAC = 1.0 - PHI_INV    # ≈ 0.382
GOLDEN_SURVIVOR_FRAC = PHI_INV        # ≈ 0.618
```

**Механизм:**
1. Популяция сортируется по fitness (убывание)
2. Верхние 38.2% = **элита** (проходят без мутации)
3. Верхние 61.8% = **выжившие** (пул родителей для мутантов)
4. Остальные 38.2% = **заменяются** мутированными потомками выживших

**Математическое обоснование:**

\[
\text{elite} = 1 - \varphi^{-1} = \varphi^{-2} \approx 0.382
\]
\[
\text{survivors} = \varphi^{-1} \approx 0.618
\]
\[
\text{elite} + \text{replaced} = 1.0
\]
\[
\frac{\text{survivors}}{\text{elite}} = \frac{\varphi^{-1}}{\varphi^{-2}} = \varphi
\]

Золотое сечение создаёт баланс между exploitation (элита) и exploration (мутанты), аналогичный термодинамическому равновесию B-ДНК (минимум свободной энергии при максимальной информационной ёмкости).

### 2.6 Апоптоз / Самовосстановление (`mutate.py`, `run_generations.py`)

**Параметры:**
```python
APOPTOSIS_THRESHOLD = 0.01
APOPTOSIS_PATIENCE = 5
```

**Механизм:**

\[
\text{discrimination}(\text{acid}) = \frac{1}{|S|} \sum_{s \in S} \left( \max_i \text{acid}(s, i) - \min_i \text{acid}(s, i) \right)
\]

Если \(\text{discrimination} < 0.01\) в течение 5 поколений подряд → **регенерация acid** (случайная реинициализация).

**Связь с исследованием:** Прямая аналогия с геномным механизмом контрольных сумм. Перес описывает: «если пропорции кодонов нарушены более чем на ~0.1%, клетка может инициировать апоптоз». В BDC threshold 0.01 (1%) на дискриминацию acid — функционально аналогичный механизм, но на уровне популяционной динамики, а не репликации.

### 2.7 3-кислотная структура генома

**ДНК-аналогия:**

| BDC Acid | Функция | ДНК-аналог | Мутабельность |
|---|---|---|---|
| **Acid_A** | Частотный prior + token biases | Backbone (стабильность) | σ₀ · φ⁻² |
| **Acid_T** | Прямой контекст (bigram) | Forward strand reading | σ₀ · φ⁻¹ |
| **Acid_G** | Обратный контекст (reverse bigram) | Reverse complement | σ₀ · 1.0 |
| **Backbone** | Аддитивная интеграция + conflict flag | Сахарофосфатный остов | — (не мутирует) |

**Формула скоринга (ClozeGenomeV2):**

\[
\text{score}[i] = \underbrace{b_i + w_f \cdot f_i}_{\text{Acid\_A}} + \underbrace{w_b \cdot \text{bigram}_i}_{\text{Acid\_T}} + \underbrace{w_r \cdot \text{rev\_bigram}_i}_{\text{Acid\_G}}
\]

Где:
- \(b_i\) — token bias для кандидата i
- \(w_f \cdot f_i\) — взвешенная нормализованная частота
- \(w_b \cdot \text{bigram}_i\) — взвешенный bigram score
- \(w_r \cdot \text{rev\_bigram}_i\) — взвешенный reverse bigram score

### 2.8 Quaternary Fitness Model (`fitness/quaternary_fitness.py`)

**Четверичная логика (не-φ, но ДНК-inspired):**

| Состояние | Код | ДНК-аналог |
|---|---|---|
| T (True) | f⁺ | Правильное сворачивание |
| F (False) | f⁻ | Мутация/ошибка |
| MY (Maybe-Yes) | f⁰ | Нейтральный полиморфизм |
| MN (Maybe-No) | f? | Неопределённость |

**Формула fitness:**

\[
\phi_{\text{fitness}} = \alpha \cdot f^+ - \beta \cdot f^- - \gamma \cdot f^0 - \delta \cdot f^?
\]

Веса по умолчанию: α=1.0, β=1.2, γ=0.3, δ=0.6.

**Variance-triggered transfer (к f?):**

\[
\text{if } \text{Var}(scores) > \theta_{\text{var}}: \quad f^?, f^+, f^-, f^0 \leftarrow \text{redistribute}(\text{shift}=\min(0.2, \text{Var} - \theta))
\]

### 2.9 Bigram-feature нормализация (`task.py`)

```python
bigram_scores = [count(prev, cand_i) / (Σ_j count(prev, cand_j) + 1)]
```

\[
\text{bigram\_score}[i] = \frac{C(\text{prev}, \text{cand}_i)}{\sum_{j} C(\text{prev}, \text{cand}_j) + 1}
\]

Аддитивная единица (+1) предотвращает деление на ноль и выполняет функцию Laplace smoothing.

### 2.10 Complexity и Fitness (`evaluate.py`)

**Regime B (canonical):**

\[
\text{complexity} = \frac{1}{K} \sum_{k=1}^{K} |w_k|
\]

\[
\text{penalty} = \lambda \cdot \text{complexity}, \quad \lambda = 0.01
\]

\[
\text{fitness}_{\text{hard}} = \text{accuracy} - \text{penalty}
\]

\[
\text{fitness}_{\text{soft}} = \frac{1}{|S|} \sum_{s \in S} P(\text{target}_s) - \text{penalty}
\]

Где \(P(\text{target})\) — softmax-вероятность правильного токена:

\[
P(k) = \frac{\exp(\text{score}_k - \max_j \text{score}_j)}{\sum_j \exp(\text{score}_j - \max_j \text{score}_j)}
\]

### 2.11 Conflict Flag (Backbone monitor)

\[
\text{var\_acid} = \frac{1}{K} \sum_{i=1}^{K} \left[ \frac{1}{3} \sum_{a \in \{A,T,G\}} (\text{acid}_a[i] - \bar{\text{acid}}[i])^2 \right]
\]

\[
\text{conflict\_flag} = \begin{cases} 1 & \text{if var\_acid} > \theta \\ 0 & \text{otherwise} \end{cases}
\]

Где \(\theta = 0.05\) (default).

### 2.12 Functional Diversity

\[
D_{\text{func}} = \frac{|\{\text{unique prediction vectors}\}|}{|\text{population}|}
\]

Каждый геном проецируется на вектор предсказаний по probe-подмножеству samples. D = 1.0 означает полное разнообразие, D → 0 означает коллапс.

### 2.13 Chi-Square Uniformity Test (`fibonacci.py`)

\[
\chi^2 = \sum_{i=1}^{k} \frac{(O_i - E_i)^2}{E_i}, \quad E_i = \frac{N}{k}
\]

P-value через Wilson-Hilferty approximation:

\[
z = \frac{\left(\frac{\chi^2}{k-1}\right)^{1/3} - \left(1 - \frac{2}{9(k-1)}\right)}{\sqrt{\frac{2}{9(k-1)}}}
\]

\[
p = \frac{1}{2} \text{erfc}\left(\frac{z}{\sqrt{2}}\right)
\]

## Часть IV. Новые предложения

### Предложение 1: φ-контрольные суммы генома (Genome Integrity Checksum)

**Источник:** Секции «Геномная контрольная сумма» и «ДНК-супракод» исследования.

**Идея:** Геном BDC-агента (ClozeGenomeV2) можно проверять на «φ-целостность» — отклонение баланса acid-каналов от ожидаемых пропорций. Если баланс нарушен сверх порога → усиленная мутация или регенерация (расширение текущего апоптоза).

**Формализация:**

\[
R_{\text{acid}} = \frac{|w_{\text{Acid\_T}}|}{|w_{\text{Acid\_A}}| + |w_{\text{Acid\_T}}| + \epsilon}
\]

Ожидаемое значение при φ-балансе:

\[
R_{\text{expected}} = \frac{\varphi^{-1}}{\varphi^{-2} + \varphi^{-1}} = \frac{\varphi^{-1}}{\varphi^{-1}(1 + \varphi^{-1})} = \frac{1}{1 + \varphi^{-1}} = \frac{1}{\varphi^{-1} + 1} = \varphi^{-1} \approx 0.618
\]

Deviation metric:

\[
\Delta_{\varphi} = |R_{\text{acid}} - \varphi^{-1}|
\]

Если \(\Delta_{\varphi} > \theta_{\varphi}\) в течение P поколений → триггер integrity repair.

**Инженерная оценка:**
- Порог \(\theta_{\varphi}\) — гиперпараметр (предложение: начать с 0.3, калибровать)
- «0.1% из исследования» — **UNVERIFIED** как универсальная константа; использовать только как стартовую точку для экспериментальной калибровки
- Метрика логируется, решение о repair основывается на L0-данных

**Уровень изменения:** R1 (новый метод, не меняет hypothesis)

### Предложение 2: FFT-спектральная диагностика дрейфа эволюции

**Источник:** Секция «Спектральный анализ и универсальные закономерности» исследования.

**Идея:** Применить спектральный анализ (FFT) к временным рядам fitness и acid-discrimination по поколениям для раннего обнаружения:
- Вырожденных периодичностей (oscillation collapse)
- Монотонного дрейфа (застревание в локальном оптимуме)
- Резкого спектрального сдвига (смена режима эволюции)

**Формализация:**

Пусть \(\{f_t\}_{t=1}^{G}\) — ряд mean_fitness по поколениям. FFT:

\[
F(k) = \sum_{t=0}^{G-1} f_t \cdot e^{-2\pi i k t / G}
\]

Спектральная энтропия:

\[
H_{\text{spec}} = -\sum_{k} p_k \ln p_k, \quad p_k = \frac{|F(k)|^2}{\sum_j |F(j)|^2}
\]

- Высокая \(H_{\text{spec}}\) → здоровая эволюция (разнообразные частоты)
- Низкая \(H_{\text{spec}}\) → доминирует одна частота (oscillation или stagnation)

**Реализация:** Пост-процессинг metrics.csv, не требует изменения core code.

**Уровень изменения:** R0 (анализ, не влияет на результаты)

### Предложение 4: Нуклеосомная память (Nucleosome-inspired Chunking)

**Источник:** Секции «Иерархическая организация хроматина» и «10n / 10n+5 паттерны».

**Идея:** Организовать будущую долговременную память агента по аналогии с нуклеосомами:

- **Chunk size ≈ F(n):** Единицы памяти хранят ≈144 элемента (F(12) ≈ 147 bp нуклеосомы)
- **Access pattern 10n (compact):** «Архивные» знания — плотно упакованы, доступны только по прямому запросу
- **Access pattern 10n+5 (open):** «Рабочие» знания — быстрый доступ, активное обновление

**Формализация доступа:**

\[
\text{accessibility}(chunk) = \begin{cases}
\text{direct\_access} & \text{if } \text{index} \mod 10 \in \{5, 15, 25, ...\} \\
\text{search\_required} & \text{if } \text{index} \mod 10 = 0
\end{cases}
\]

**Уровень изменения:** R2 (новая гипотеза для будущей архитектуры памяти, не текущий приоритет)

### Предложение 5: φ-scaled Context Windows для Skip-Bigram (TASK-1606)

**Источник:** Фрактальное самоподобие ДНК на разных уровнях + текущий TASK-1606.

**Идея:** В текущем TASK-1606 skip-bigram использует расстояние skip=2 (prev_prev, next_next). Можно расширить до Fibonacci-лестницы расстояний: skip ∈ {1, 2, 3, 5, 8} для захвата контекста на разных масштабах.

**Формализация:**

\[
\text{score}[i] = \sum_{d \in \mathcal{F}} w_d \cdot \text{context\_score}(d, i)
\]

Где \(\mathcal{F} = \{1, 2, 3, 5, 8\}\) — Fibonacci-расстояния.

**Мутация весов:** Каждый уровень расстояния мутирует с σ, масштабированным по φ⁻ⁿ:
- skip=1: σ₀ · φ⁻¹ (ближний контекст — консервативный)
- skip=2: σ₀ · φ⁰
- skip=3: σ₀ · φ¹
- skip=5: σ₀ · φ²
- skip=8: σ₀ · φ³

**Предупреждение:** Это **не** замена текущему TASK-1606. Это будущее расширение после завершения TASK-1606 и анализа его результатов. Не применять до получения L0-результатов диагностического sweep.

**Уровень изменения:** R1 (новый метод, требует отдельный TASK)

### Предложение 6: Квазикристаллическая популяционная топология

**Источник:** Секция «Биологические квазикристаллические структуры» — квазикристаллы имеют дальний порядок без трансляционной симметрии.

**Идея:** Заменить «плоский» массив популяции на структуру с neighbourhood-отношениями, основанными на Penrose tiling. Мутация происходит только между «соседями» в quasi-crystal lattice, что создаёт:
- Дальний порядок (глобальная информация распространяется)
- Локальные кластеры (сохранение разнообразия)
- Отсутствие периодичности (нет застревания в циклах)

**Инженерная оценка:**
- Высокая сложность реализации
- Неочевидный выигрыш при текущем размере популяции (100)
- Возможно актуально при P > 1000

**Уровень изменения:** R2 (гипотеза, требует отдельное исследование)

### Предложение 7: Variance-Triggered φ-Rebalance (Extension of Apoptosis)

**Источник:** Геномные контрольные суммы + текущий conflict_flag.

**Идея:** Расширить текущий механизм апоптоза: вместо бинарного «мёртв/жив» для каждого acid, ввести непрерывный φ-баланс между acids.

**Формализация:**

\[
\text{balance}(g) = \frac{\text{acid\_T\_disc}(g)}{\text{acid\_A\_disc}(g) + \epsilon}
\]

Ожидаемый φ-баланс: balance ≈ φ⁻¹/φ⁻² = φ ≈ 1.618.

Если balance отклоняется за пределы [1.0, 2.618] (φ ± φ⁻¹):

\[
\sigma_{\text{corrected}} = \sigma_{\text{acid}} \cdot \left(1 + \alpha \cdot |\text{balance} - \varphi| \right)
\]

Это «мягкий» корректирующий механизм вместо «жёсткого» апоптоза.

**Уровень изменения:** R1 (модификация метода мутации)

# BDC Paid Pilot Policy

## Policy Intent
Paid pilot is the default path for qualified prospects. Free pilot is an exception used only for strategic proof gaps.

## Qualification Gate
Prospect must satisfy:
- baseline freeze readiness
- architecture-sensitivity evidence
- measurable target metric
- stakeholder owner assigned
- restricted BDC scope acceptance

## Decision Rule
- High qualification + medium/high urgency -> paid pilot default.
- Medium qualification + high strategic value -> conditional paid pilot with scoped discount.
- Low qualification -> no pilot until prerequisites are closed.

## Pricing Logic
- Starter: fastest validation and one workflow.
- Standard: instrumentation and stronger comparison package.
- Advanced: multi-team advisory and hybrid search depth.

## Trust Posture
- Claims remain restricted to validated scope.
- No promise of universal optimization.
- Negative findings are retained in reporting.

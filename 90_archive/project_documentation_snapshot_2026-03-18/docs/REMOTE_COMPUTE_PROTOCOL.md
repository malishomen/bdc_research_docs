# Remote Compute Protocol (BDC Queen)

## Purpose

Use `bdc-queen` (`bdc@192.168.1.100`) as the compute island for heavy experiments while preserving reproducibility and compact artifact transfer.

## Pre-Run Checks

1. Run healthcheck:
   - Windows: `pwsh -File scripts/ops/queen_healthcheck.ps1`
   - Linux/macOS: `bash scripts/ops/queen_healthcheck.sh`
2. Confirm branch and commit locally:
   - `git rev-parse --short HEAD`
3. Record exact run command in task report (`RUN_COMMANDS.md` / brief report).

## Running An Experiment Remotely

Use:

```powershell
pwsh -File scripts/ops/queen_run_experiment.ps1 `
  -Commit <git-commit> `
  -RunCommand "<command>" `
  -OutRoot "<remote-results-root>"
```

Example smoke for EDP1:

```powershell
pwsh -File scripts/ops/queen_run_experiment.ps1 `
  -Commit e8152d6 `
  -RunCommand "python3 -m evolution.edp1_symbolic.run_generations --out_dir results/remote_smoke_task_hive_001 --genome_version v1 --seed 1337 --generations 5 --population 20" `
  -OutRoot "results/remote_smoke_task_hive_001"
```

What the dispatcher does:

1. SSH to queen.
2. `git fetch` + detached checkout to requested commit.
3. Execute the supplied command.
4. Pack and pull compact artifacts only:
   - `summary.json`
   - `aggregates/*`
   - `REPORT.md`, `*BRIEF_REPORT.md`
5. Store local pull in `results/remote_compact/<run_id>/`.

## Artifact Rules

- Do not commit raw remote `results/` trees.
- Commit only compact analysis artifacts if needed (reports, small JSON summaries).
- Never commit tokens/secrets.

## Ingress/Health Notes

- For `viz.bdc-hive.com/dist/LATEST.json`, use `GET` (not `HEAD`):
  - `GET` -> expected `200`
  - `HEAD` -> expected `405`
- `queen.bdc-hive.com` should remain Access-protected (`403` without Access headers).

## Applying To exp_0400 / exp_0500

- Keep experiment command explicit in `-RunCommand`.
- Write exact command and commit into task report.
- Pull only compact summary artifacts and attach links in report.

Example smoke for exp_0400 (Cloze symbolic):

```powershell
pwsh -File scripts/ops/queen_run_experiment.ps1 `
  -Commit <commit> `
  -RunCommand "python3 scripts/edp1/run_cloze_exp_0400.py --smoke --seeds 2 --generations 10 --population 20 --out_root results/edp1_exp0400_cloze_remote_smoke" `
  -OutRoot "results/edp1_exp0400_cloze_remote_smoke"
```

Expected compact pullback:
- `seed_*/summary.json`
- `aggregates/summary.json`
- task brief report if generated by the run wrapper.

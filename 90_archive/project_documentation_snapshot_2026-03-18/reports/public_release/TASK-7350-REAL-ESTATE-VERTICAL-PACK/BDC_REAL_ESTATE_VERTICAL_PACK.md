# BDC Real Estate Vertical Pack

## Vertical Position
BDC Designer supports real-estate workflow architecture decisions for property intake, matching, valuation support, and offer-generation review.

## Target Use Cases
- Property intake -> enrichment -> matching -> advisor review
- Buyer qualification -> listing shortlist -> risk check -> offer draft
- Valuation support -> compliance verifier -> final recommendation gate

## Typical Pilot Structure
1. Freeze baseline throughput and quality metrics.
2. Run BDC CLI recommendation.
3. Apply hybrid refinement for role allocation and strategy.
4. Compare baseline vs BDC vs hybrid outcomes.
5. Package proof for internal decision and external credibility.

## Scope Safety
Restricted BDC scope applies; no universal market-performance guarantees.
